# Neve Child Theme

## التثبيت والتفعيل

### الخطوات:

1. **تفعيل الثيم الابن:**
   - اذهب إلى لوحة التحكم WordPress
   - اذهب إلى: المظهر > القوالب (Appearance > Themes)
   - ابحث عن "Neve Child"
   - اضغط على "تفعيل" (Activate)

2. **التحقق من التفعيل:**
   - تأكد من أن الثيم النشط هو "Neve Child" وليس "Neve"

## المشاكل التي تم حلها:

### 1. مشكلة الكونتينر (العناصر رايحة ناحية الشمال)
- تم إضافة CSS لتوسيط الكونتينر تلقائياً
- تم ضبط `margin-left: auto` و `margin-right: auto`
- تم ضبط عرض الكونتينر ليكون responsive

### 2. مشكلة الفوتر المكرر
- تم إنشاء ملف `footer.php` مخصص
- تم إضافة كود لمنع تكرار الفوتر باستخدام `static $footer_rendered`
- تم إضافة CSS لإخفاء أي فوتر مكرر

## الملفات المضافة:

- `style.css` - ملف الأنماط الرئيسي مع التصحيحات
- `functions.php` - وظائف الثيم الابن
- `footer.php` - قالب الفوتر المعدل
- `README.md` - هذا الملف

## التخصيصات الإضافية:

إذا كنت تريد تعديلات إضافية، يمكنك:

1. **تعديل عرض الكونتينر:**
   - افتح `style.css`
   - عدل قيمة `max-width` في الكلاس `.container`

2. **تعديل المحاذاة:**
   - افتح `style.css`
   - عدل `text-align` و `direction` حسب الحاجة

3. **إضافة CSS مخصص:**
   - أضف الكود في نهاية ملف `style.css`

## ملاحظات مهمة:

- لا تعدل على ملفات الثيم الأب (Neve) مباشرة
- كل التعديلات يجب أن تكون في الثيم الابن (Neve Child)
- عند تحديث الثيم الأب، تعديلاتك ستبقى آمنة

## الدعم:

إذا واجهت أي مشاكل:
1. تأكد من تفعيل الثيم الابن
2. امسح الكاش (Cache) من المتصفح
3. امسح كاش WordPress إذا كنت تستخدم plugin للكاش
